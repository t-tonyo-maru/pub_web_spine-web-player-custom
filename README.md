# pub_web_spine-web-player-custom

## 概要

本アプリは [Spine Web Player](http://ja.esotericsoftware.com/spine-player)をベースにした Web アプリです。  
Web ページ上で、Spine アニメーションのプレビュー。および、一枚の html への変換ができます。
